Conexiones

Cómo entran y salen tus datos: sube un Excel/CSV, o conecta tu propio sistema por API.

Importar desde Excel / CSV

Sube un archivo .xlsx / .xls / .csv con tus puntos o trabajadores. Te dejamos asignar cada columna antes de importar.

¿Cómo preparar tu Excel?

No necesitas cabeceras exactas ni saber de informática: al subir el archivo eliges qué columna es cada cosa. Así puede verse tu Excel — puedes copiar estos ejemplos:

Nombre obligatorio Dirección obligatorio Teléfono del trabajador opcional
Silpo KhreshchatykKhreshchatyk St 15+380671112233
ATB PodilVerkhnii Val St 24+380677654321

Si añades el teléfono del trabajador, la parada queda asignada a esa persona automáticamente. Las coordenadas se rellenan solas en el primer check-in.

Nombre obligatorio Teléfono obligatorio
Ivan Petrenko+380671112233
Olena Kovalenko+380677654321

El teléfono es la clave: cuando el trabajador comparte su número en el bot, se enlaza con esta ficha. Escríbelo con prefijo de país (ej. +34…).

Conexión por API (conector)

El cliente empuja su catálogo y lee las visitas por API. Se autentica con la cabecera X-API-Key. Estado: …

Esta es la clave única de tu empresa. Entrégasela a quien integre tu sistema. Si la regeneras, la anterior deja de funcionar de inmediato.

{
  "workers": [
    { "name": "Ivan Petrenko", "phone": "+380671112233" },
    { "name": "Olena Kovalenko", "phone": "+380677654321" }
  ]
}
{
  "points": [
    { "id": "P1", "name": "Silpo Khreshchatyk", "address": "Khreshchatyk St 15, Kyiv", "workerPhone": "+380671112233" },
    { "name": "ATB Podil", "address": "Verkhnii Val St 24, Kyiv", "workerPhone": "+380677654321" }
  ]
}

workerPhone (opcional) enlaza el punto con su trabajador asignado. Las coordenadas se rellenan solas en el primer check-in de cada punto.

Bot de Telegram …
Comprobando…

StarX usa un único bot compartido, ya configurado por la plataforma — tú solo lo enciendes o apagas aquí, no hay tokens que gestionar.

  1. Carga a tus trabajadores con su teléfono (en Trabajadores o por importación/API).
  2. El trabajador abre el bot y pulsa /start (o usa el enlace de abajo).
  3. Comparte su número de teléfono y queda enlazado a tu empresa automáticamente.
  4. Pulsa /route, elige una parada y hace su check-in con ubicación y foto.

Compártelo por WhatsApp/Telegram. Quien lo abra se registrará en tu empresa (código …).

Si el botón Encender bot aparece deshabilitado, el operador de la plataforma aún no ha configurado el token del bot en el servidor.
